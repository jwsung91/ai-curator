## 💡 오늘의 관찰

- 물리 세계를 인지하는 OAK 카메라 투자 유치와 대규모 AI 팩토리 가동은 로보틱스의 피지컬 AI가 실질적인 양산 및 산업 확장 단계에 진입했음을 보여줍니다.
- ROS 2 패키지 대규모 업데이트와 PlotJuggler 4의 다중 모달 데이터 시각화 지원은 로봇 런타임 환경의 안정성과 복잡한 실시간 데이터 분석 역량을 동시에 강화하고 있습니다.
- MCP 클라우드 서비스와 에이전트 메모리 기반 SDK 고도화 흐름은 복잡해지는 코딩 협업 에이전트 인프라를 확장하여 로보틱스 시스템과의 지능형 제어 연계를 더 용이하게 만듭니다.

---

## 🤖 로보틱스

- **PlotJuggler 4 (beta)**: 다중 모달 데이터를 지원하는 시각화 도구의 초기 프리뷰 버전 출시 [<a href="#ref-1">1</a>]
- **kc_vision**: ROS 2 Jazzy 및 Gazebo Harmonic 환경에서 SLAM, Nav2, GPS 기반 boustrophedon 경로 계획을 지원하는 오픈소스 지상 로봇 프로젝트 [<a href="#ref-2">2</a>]
- **Mastering NERO**: CAN 버스 통신을 이용해 AgileX NERO 7-DoF 로봇 팔을 마스터-팔로워 구조로 구성하는 듀얼 암 튜토리얼 [<a href="#ref-3">3</a>]
- **Gazebo Audio Features**: SDF v1.12 사양 하의 World 오디오 속성 API 활용에 대한 커뮤니티 문의 [<a href="#ref-4">4</a>]
- **ROS 2 Humble Hawksbill 업데이트**: 147개의 새로운 패키지와 618개의 업데이트 패키지 동기화 발표 [<a href="#ref-5">5</a>]
- **rclcpp 33.0.2**: ROS 2 C++ 클라이언트 라이브러리 rclcpp의 패치 릴리스 [<a href="#ref-6">6</a>]
- **Fast DDS v3.6.2.0**: 리포지토리 구성(.repos)을 업데이트한 신규 DDS 릴리스 [<a href="#ref-7">7</a>]
- **rclpy 11.0.2**: ROS 2 Python 클라이언트 라이브러리 rclpy의 패치 릴리스 [<a href="#ref-8">8</a>]

## ✨ AI

- **DSPy를 통한 SQL 프롬프트 최적화**: Datasette Agent의 SQL 시스템 프롬프트를 평가하고 개선하기 위해 DSPy 프레임워크를 적용한 사례 연구 [<a href="#ref-9">9</a>]
- **Understand to participate**: 복잡해지는 코딩 에이전트 개발 흐름에 맞춰 협업의 필요성을 강조한 에이전트 상호작용 개념 제시 [<a href="#ref-10">10</a>]
- **Manufact (YC S25)**: 클라우드 기반으로 모델 컨텍스트 프로토콜(MCP) 서버 환경을 제공하는 MCP Cloud 서비스 론칭 [<a href="#ref-11">11</a>]
- **Claude-real-video**: 다양한 LLM에서 비디오 스트림 데이터를 분석하고 이해할 수 있도록 지원하는 오픈소스 프로젝트 [<a href="#ref-12">12</a>]
- **slopo**: 임베딩 모델을 활용하여 유사도가 일치하지 않는 비정형 코드 중복을 감지하는 명령줄 인터페이스(CLI) 도구 [<a href="#ref-13">13</a>]
- **Anthropic SDK v0.116.0**: agent-memory-2026-07-22 베타 헤더 기능을 포함한 파이썬 SDK 업데이트 [<a href="#ref-14">14</a>]
