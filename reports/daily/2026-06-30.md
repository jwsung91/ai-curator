## 💡 오늘의 관찰

- LLM과 에이전트 기술의 발달이 로봇 인터페이스 및 개발 워크플로우로 스며들고 있습니다.
- 로봇 개발자들을 위해 CAD 변환, 비헤이비어 트리 편집 등 수작업이 많던 작업을 자동화하거나 IDE 내에서 일원화하려는 에코시스템의 노력이 두드러집니다.
- 산업 현장에서는 휴머노이드 로봇의 실제 생산 라인 투입과 우주 환경 등 극한 영역으로의 하드웨어 진출이 구체화되고 있습니다.
- 엔터프라이즈 레벨의 AI 에이전트 운영을 위해 클라우드 하드웨어 최적화와 대규모 파트너십이 한층 강화되고 있습니다.

---

## 🤖 로보틱스

- **Guidelines for AI-Assisted Contributions in the ROS Project**: ROS PMC가 LLM 기반 PR 증가에 대응하여 OSRF의 생성형 도구 가이드라인을 확장 적용하기로 결정했습니다 [<a href="#ref-1">1</a>]
- **Ros2-dev-suite**: 파라미터 튜너, 서비스/토픽/액션 호출기 등을 통합한 ROS 2 개발용 VS Code 익스텐션을 공개했습니다 [<a href="#ref-2">2</a>]
- **Nav2 BT Editor**: XML을 직접 수정하지 않고 VS Code 내에서 Nav2 비헤이비어 트리를 편집하고 미리 볼 수 있는 도구입니다 [<a href="#ref-3">3</a>]
- **A Standard Interface from LLM to Robot Actions**: LLM 기반 인지 시스템과 ROS/2 로봇 간의 표준 인터페이스 제안인 ClawsJoy Robotics가 발표되었습니다 [<a href="#ref-4">4</a>]
- **Cad to urdf conversion**: 복잡한 CAD 파일을 URDF로 간편하게 변환할 수 있는 웹 기반 도구인 Jointly가 런칭되었습니다 [<a href="#ref-5">5</a>]
- **NVIDIA Jetson in Lunar Orbit**: Firefly Aerospace가 달 궤도에서 최초로 NVIDIA Jetson 보드를 동작시키는 데 성공했습니다 [<a href="#ref-6">6</a>]

## ✨ AI

- **Ornith-1.0**: DeepReinforce가 출시한 에이전트 코딩 및 자가 개선(Self-Scaffolding)을 지원하는 오픈 소스 LLM입니다 [<a href="#ref-7">7</a>] [<a href="#ref-8">8</a>] [<a href="#ref-9">9</a>]
- **Quoting Jon Udell**: 에이전트가 검토하기 어려울 정도로 대량의 PR을 생성하는 문제를 지적하며 인간 중심의 제어 루프 유지를 강조했습니다 [<a href="#ref-10">10</a>]
- **Anthropic SDK v0.113.0**: 20260318 웹 패치 및 지원 도구 기능 등이 추가된 파이썬 SDK 업데이트가 배포되었습니다 [<a href="#ref-11">11</a>]
- **Ollama v0.31.0**: Hermes 데스크톱 최소 버전 체크 기능 등이 포함된 신규 릴리스가 공개되었습니다 [<a href="#ref-12">12</a>]

## 📈 트렌드

- **Mapping Europe’s AI Workforce Opportunity**: AI가 유럽 연합 국가 내 일자리의 자동화, 워크플로우 변화에 미치는 영향을 분석한 OpenAI 보고서가 발표되었습니다 [<a href="#ref-13">13</a>]
- **HP Inc. Frontier Partnership**: HP가 OpenAI와의 Frontier 파트너십을 확장하여 소프트웨어 개발 및 기업 운영 전반에 AI를 도입합니다 [<a href="#ref-14">14</a>]
- **BMW Group Figure 03 Deployment**: BMW가 Figure 02의 생산 라인 테스트 성공에 이어 최신 Figure 03 휴머노이드 로봇을 배치합니다 [<a href="#ref-15">15</a>]
- **MBody AI Expansion**: 하드웨어 불가지론적 오케스트레이터 플랫폼을 운영하는 MBody AI가 미국 11개 주 및 캐나다로 서비스 영역을 확장했습니다 [<a href="#ref-16">16</a>]
- **Hypoid Gearing Advantages**: 기계적 효율성과 부하 용량의 균형을 제공하는 하이포이드 기어의 로봇 설계상 장점이 소개되었습니다 [<a href="#ref-17">17</a>]
- **Claude Meets Blackwell Ultra**: Anthropic의 Claude 모델이 Microsoft Azure 내 NVIDIA GB300 Blackwell Ultra GPU 기반에서 공식 지원됩니다 [<a href="#ref-18">18</a>]
