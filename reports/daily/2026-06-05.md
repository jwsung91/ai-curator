## 💡 오늘의 관찰

- AI 에이전트가 별도의 미들웨어 브릿지(rosbridge) 없이 ROS2와 직접 상호작용할 수 있는 네이티브 MCP 서버의 등장은 로보틱스 시스템에 LLM을 이식하는 아키텍처를 크게 간소화합니다.
- 산업용 물류 로봇 분야(Amazon Proteus)의 자연어 제어 역량 도입은 단순한 기능 업데이트를 넘어 비숙련 작업자의 온보딩 비용을 획기적으로 낮추는 실무적 트렌드입니다.
- 로컬 LLM 구동을 위한 최적화 백엔드(KVarN, Ollama)와 클라우드 모델 관리자(LiteLLM)의 고도화는 엣지 디바이스 및 하이브리드 로보틱스 환경의 추론 안정성을 배가시킵니다.
- 수술용 로봇의 원격 지원 및 군용 응급 치료 시도 등 물리 로봇의 작동 반경이 특수 도메인으로 더욱 넓어지고 있으며, 범용 제어 모델 시장의 자본 유치가 이에 속도를 더하고 있습니다.

---

## 🤖 로보틱스

- **Sowbot / Feldfreund DevKit ROS**: 농업용 정밀 파종 로봇 개발을 위한 오픈소스 오픈하드웨어 ROS2 스택입니다. [<a href="#ref-1">1</a>]
- **2D-Scan Merger for ROS2**: 여러 개의 2D 레이저 스캔 데이터를 하나로 병합해 주는 ROS2 패키지가 공개되었습니다. [<a href="#ref-2">2</a>]
- **Amazing ROS 2 MCP**: rosbridge 없이 rclpy를 통해 AI 에이전트가 ROS2 시스템과 직접 상호작용할 수 있도록 지원하는 네이티브 Model Context Protocol(MCP) 서버입니다. [<a href="#ref-3">3</a>]
- **Wrench observation Isaaclab**: IsaacLab 시뮬레이션 환경 내에서 힘-토크 센서와 유사한 랜치(Wrench) 데이터를 획득하는 구현법에 대해 논의가 진행되었습니다. [<a href="#ref-4">4</a>]
- **Gazebo gz-sim10_10.4.0**: 대화형 시뮬레이션을 원활히 지원하기 위해 Arjo Chakravarty와 Steve Peters 등이 참여한 10.4.0 릴리스 준비가 완료되었습니다. [<a href="#ref-5">5</a>]

## ✨ AI

- **Endava software delivery**: ChatGPT Enterprise와 Codex를 활용하여 소프트웨어 인도 과정을 최적화하고 에이전트 기반 개발 문화를 정착시키는 사례가 공개되었습니다. [<a href="#ref-6">6</a>]
- **Dreaming (ChatGPT Memory)**: 대화의 맥락 유지와 맞춤형 어시스턴트 역할을 강화하기 위해 ChatGPT에 새로운 메모리 시스템이 업데이트되었습니다. [<a href="#ref-7">7</a>]
- **defending-code-reference-harness**: Anthropic이 AI 기반 코드 취약점 발견 및 방어 프레임워크를 오픈소스로 배포했습니다. [<a href="#ref-8">8</a>]
- **KVarN**: Huawei 연구진이 vLLM 백엔드에서 KV-cache 양자화를 네이티브하게 제공할 수 있도록 개발한 라이브러리입니다. [<a href="#ref-9">9</a>]
- **MCP Servers 2026.6.3**: Model Context Protocol 공식 서버 저장소의 신규 2026.6.3 버전이 배포되었습니다. [<a href="#ref-10">10</a>]
- **Ollama v0.30.5**: gemma4:12b 모델 사용 시 발생하는 부동소수점 크래시를 패치하고 윈도우 설치 환경을 정비했습니다. [<a href="#ref-11">11</a>]
- **LiteLLM v1.87.1**: Docker 이미지 서명 검증 절차를 도입하고 라이브러리 보안 체계를 개선한 버전이 출시되었습니다. [<a href="#ref-12">12</a>]

## 📈 트렌드

- **SS Innovations 수술용 로봇**: 분쟁 지역이나 극한 상황에서 원격 응급 수술을 가상 지원할 수 있는 비행 가능한 수술용 로봇 프로젝트의 극복 과제를 다루었습니다. [<a href="#ref-13">13</a>]
- **Amazon Proteus**: 아마존이 유럽 물류 기지에 모바일 로봇 Proteus를 전개하며 특별한 통제 명령 없이 기계를 직관적으로 지시할 수 있는 자연어 인터페이스 역량을 수혈했습니다. [<a href="#ref-14">14</a>]
- **Generalist 4억 달러 유치**: 범용 AI 제어 모델의 작업 성공률을 최대 99% 수준까지 끌어올리기 위한 대규모 투자 라운드를 마쳤습니다. [<a href="#ref-15">15</a>]
